# BACSA-Suspect-Database

A forensic database containing DNA profiles and fingerprint records for 15 individuals identified as persons of interest in the BACSA HACKS break-in. DNA samples were collected via court-ordered forensic swabbing; fingerprints were recorded under standard tenprint booking procedure. Cross-reference entries against crime scene evidence repositories to establish identity matches.

```Suspect_DNA.fasta``` is a fasta file containing all 15 DNA profiles. ```Suspect_Fingerprints``` is the directory containing BMP fingerprint files, each one named after each suspect.
